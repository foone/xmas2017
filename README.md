# xmas2017

An animated Christmas Card made for 2017.

Requirements:

* DOS 2.0+
* CGA graphics or better

The EXE can be created using QuickBasic 4.5.

The program requires the SCREEN1.BSV & SCREEN2.BSV files which can be created using img2bsave.py
